c++primer 学习笔记（四）--含有无符号类型的表达式

Sep 17, 2017 | 阅读

无符号类型的数据不可能为负值，所以当一个无符号类型的数据在操作中将负数转换成无符号数进行运算时，结果等于这个负数加上无符号数的模。当从无符号数中减去一个值时 ，不管这个值是不是无符号数，我们都必须确保结果不会是一个负值。
在写循环的时候，无符号数不会小于零的这一事实会对循环写法产生很大的影响。所以在循环中使用到无符号数时，一定要注意循环的写法。

这个循环将会变成一个死循环，因为当运行到u=0时，再减一使u=-1,此时-1不满足无符号数的要求，会自动转换成一个合法的无符号数，即结果等于这个负数加上无符号数的模。

使用这个while循环，先将数字–后，再输出。这样当运行到u=1,–u后输出u=0。再循环的时候，u=0不满足循环条件，不会进入循环，则不会出现u<0的情况使u不满足无符号数的要求。

值得注意的是：一定不能混用带符号类型及无符号类型，因为带符号类型会自动转换为无符号数。这样出现负数则会输出异常结果。
